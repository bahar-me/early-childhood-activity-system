# Sprint 3–8 Plan

## Sprint 3: Database and School Management

### Sprint Goal

To establish the database layer and implement school management functionality with role-based access control.

### Planned Tasks

* Design database schema using SQLAlchemy
* Create School model and relationships
* Implement school create, list, update, and delete operations
* Configure database migrations with Alembic
* Implement role-based access control for school operations
* Develop school management API endpoints
* Create backend test cases

### Expected Outputs

* School Management API
* Database relationships and migrations
* Protected endpoints with RBAC
* Backend unit tests

---
