# Project Report Summary

## Project Name
Early Childhood Activity Recommendation System

## Project Purpose
The purpose of this project is to support preschool teachers in planning activities according to teacher profiles, class profiles, age groups, and learning goals.

## Main Features
- User login and role-based authorization
- Teacher and class profile usage
- Activity planning flow
- AI-supported activity adaptation
- PDF activity report generation
- Local LLM support with Ollama
- Mobile preparation with Capacitor

## Technologies Used

### Frontend
- React
- TypeScript
- Vite
- Tailwind CSS

### Backend
- Flask
- SQLAlchemy
- JWT
- Pytest

### Artificial Intelligence
- Gemini API
- Ollama Local LLM
- Mock AI fallback

### Mobile
- Capacitor
- Android project structure

## Testing
Backend tests were prepared for login, authentication flow, school operations, and role-based authorization.

## Result
The project provides a working prototype that supports teachers in planning, adapting, and reporting preschool activities.
